Admin Panel Documentation

Admin Panel is available to admins of the website for managing various things It has following modules:

Report

Reports module can be used to generate different reports from the database available on the wbsite.

Following are the types of reports available

  • People
  • Publications
  • Books
  • Events
  • Activities
  • News
  • Announcements

Home

Home tab in the admin panel is for the updation of contents on home page

The content can be edited by clickng Edit Content button.

People

People tab is the repository of all the diffeent types of people connected with the Depatment. It lists all types of Users and an admin with an access to this module can add/update/delete users and their information.

Faculty / Inspire Faculty / Visiting Faculty

These three kind of users have following features

  • Registration with IITM Email ID
  • ADS Login
  • A Dedicated page for them where various information can be displayed
Post Doctoral Fellows / Ph.D

PDFs and Ph.Ds has following features

  • Registration with any Email ID
  • Normal Login (Password can be changed by an admin)
  • A Dedicated page for them where various information can be displayed
Retired Faculty

Retired Faculty have following features

  • Listing of faculties with their conact details
Staff

Staffs have following features

  • Listing of staffs with their official contact details

Student

Students tab is the repository for all the students categorized by programs and year. Students does not have login facility. A student can be added manually or a list of students can also be imported from a excel sheet(.xls file). A link to download a sample file for import is included beside the import button in the Students Tab

A Delete button on the top in the Students Tab will be handy if multiple/all students are selected and then deleted.

Research

Research Tab has three sub-sections

Research Area

In this tab a description about research by department can be displayed on the website.

Research Groups

Research Groups can be added/updated/deleted from this tab

Recent Publications

Recent Publication is a quick listing of publications from the faculties of the department.

Programs

Programs tab is for managing different programs offered by the depatment. A program can be added and existing programs can be updated/deleted in this tab. Programs can have many semesters and each semester can have multiple courses. All these courses can be easily added/modified.

Other Courses

In this tab courses other than those which has been listed in programs can be added.

Events

Events lists al the various events that are conducted by the Department. Events can be categorized. And new categories can be added dynamically.

After an event gets created a notification starts displaying on the Home page of website which links to the event page.

Event Page is generated based on the data entered while creating the event

Resource

Resource tab has various important resources related to the department.

Links

Important links can be displayed on this page and they can also be restricted to be seen only after logging in.

Documents

Important documens can be uploaded here which again can be restricted for logged in users if needed.

Gallery

All the images related to department can be displayed in gallery categorized into various albums.

Videos

Videos by department can be updated here.

Conference Halls

Conference halls of department can be added here which will be used in booking module.

Slider

Slider is responsible for the image slider on the Home Page. Images can be added or removed from slider and also their order of apearance can be modified.

Bookings

Booking Module lists all the bookings made by any faculties. These bookings can be cancelled by an admin from this tab. A new boking can also be done from this tab.

News/Announcements

Through this tab news or announcements can be displayed on the website. News are displayed on the home page at the right sidebar while announcement appear in the scrolling text below the logo on home page.

Contact Info

Contact Info page has a HTML editor where contact information can be updated an same thing gets reflected on the contact pag of website.

User Priveleges

Through this tab any user can be given permission to access different modules as per requirement
