Database Configurations

Any changes in the database credential by Server Administrator (Computer Center -IITM) must be updated at following locations

APP's Connection to Database

File Name & Path - /dptcms_base/app/config/database.php

Line No - 55 to 64

'mysql' => array(
    'driver'    => 'mysql',
    'host'      => 'localhost',
    'database'  => 'math',
    'username'  => 'math',
    'password'  => '*******',
    'charset'   => 'utf8',
    'collation' => 'utf8_unicode_ci',
    'prefix'    => '',
),`
BIBTEX connection to Database

File Name & Path - /dptcms_base/app/views/bibteximport.php

Line No - 12 to 22

/** MySQL database username */
@define('DB_USER', 'math');

/** MySQL database password */
@define('DB_PASSWORD', '******');

/** MySQL hostname */
@define('DB_HOST', 'localhost');

/** The name of the database */
@define('DB_NAME', 'math');

protected $domainControllers = array("10.24.3.51");
